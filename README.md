# MetaSense AI Project

## Overview
MetaSense is an AI-assisted framework to evaluate research methodology quality.

## Features
- Upload research papers
- Analyze methodology using NLP (prototype)
- Generate feedback report

## Run Instructions
1. Install dependencies:
   pip install -r requirements.txt

2. Run app:
   python app.py

3. Open browser:
   http://127.0.0.1:5000
